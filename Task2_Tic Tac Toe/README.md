# Tic-Tac-Toe AI 🎮

## 📌 Overview
This project is an AI-powered Tic-Tac-Toe game developed as **Task 2** for the **CodSoft AI Internship**. The game allows a human player to compete against an AI opponent that makes intelligent moves, creating a challenging and interactive gameplay experience.

## ✨ Features
- Human vs AI gameplay
- Intelligent AI move selection
- Interactive command-line interface
- Input validation
- Win, lose, and draw detection
- Clean and easy-to-understand Python code

## 🛠️ Technologies Used
- Python 3
